# 🧮 Menu Web de Bozha

Proyecto de práctica realizado cin **HTML5**, **CSS3** y **JavaScript** vanilla.

## 📋 Caracteristicas

    - Uso del addEventListener.
    - Diseño responsivo para moviles y escritorio.
    - Uso del LocalStore para guardar el idioma seleccionado

## 🚀 Tecnologías utilizadas

    - Flexbox para el layout.
    - Event Delegation y manipulacion del DOM.
